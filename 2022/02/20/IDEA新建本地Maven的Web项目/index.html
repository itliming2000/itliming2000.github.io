IDEA新建本地Maven的Web项目

发表于 2022-02-20 更新于 2022-03-07 分类于 Java Web Valine：
本文字数： 1.3k 阅读时长 ≈ 1 分钟

Maven作为一个重要的项目管理工具，其本身具有的【依赖管理】与【一键构建】功能，极大程度上便捷了Web项目的管理。下面，我们就来介绍如何使用IDEA新建本地Maven的Web项目。

软件版本
· IntelliJ IDEA 2020.3.2
· apache-maven-3.8.1

一、安装Maven

1.下载

maven官网下载地址：https://maven.apache.org/download.cgi ,下载最新版本的maven程序包。

2.安装

将.zip文件包，解压到D:\maven\apache-maven-3.8.1目录下:

二、配置Maven

1.设置环境变量

【控制面板\系统和安全】-【系统】

【高级系统设置】-【系统属性】-【设置环境变量】

新建系统变量MAVEN2_HOME，赋值到maven安装目录D:\maven\apache-maven-3.8.1
注：变量名可自定义，有时无法安装成功，考虑修改别名，或能成功！

新建路径变量，Path下，追加%MAVEN_HOME%\bin

2.检查安装成功

至此，基本完成maven的安装，现在运行DOS命令，检查是否安装成功！
打开命令提示符窗口，输入mvn -v，回车后显示类似如下窗口结果，即安装成功！

3.配置本地仓库

新建本地仓库

在D:\maven\目录下新建maven-repository文件夹，该文件夹作为maven的本地仓库，存储下载而来的相关.jar包。

指定本地仓库

  1. 打开D:\maven\apache-maven-3.8.1\conf\settings.xml文件
  2. 查找 代码结点<!– localRepository ，将其中指定代码移出注释区域，修改为准备好了的文件夹地址，如下。

4.修改国内镜像

由于默认情况下执行 mvn 各种命令是去国外的 mvn 官方镜像源获取需要安装的具体软件信息，所以在不使用代理、不翻墙的情况下，从国内访问国外服务器的速度相对比较慢。阿里旗下维护着一个国内 maven 镜像源，可以加速下载
