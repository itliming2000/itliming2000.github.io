【Hexo】nexT主题使用攻略基础——添加评论功能

发表于 2022-03-06 更新于 2022-03-07 分类于 Hexo Valine：
本文字数： 1.6k 阅读时长 ≈ 1 分钟

针对 Hexo 静态博客的 nexT主题 继承了多种评论系统，此处介绍两种相对好用的评论系统：Valine 和 Gitment。

一、Valine 评论系统

Valine 评论系统，此款系统不需要登录，无后端，nexT主题最新也已经支持安装。

1.1 特点

  • 基于 LeanCloud 的系统
  • 支持 MarkDown 语法
  • 无后端实现，快速高效

1.2 配置

注册 LeanCloud 系统

因为 Valine 是基于 LeanCloud 系统的，所以先在 LeanCloud 中注册账号

LeanCloud官网：https://console.leancloud.cn/login

注册登陆后，访问控制台，创建应用，选择开发版，确定创建，如下图

二、Gitment 评论系统

Gitment评论系统是基于 Github 开发的，是依靠 GitHub Issues 的评论系统，Next 主题最新也已经支持安装。

安装前提：更新 Next 主题（5.1.2 主题及以上）

2.1 注册 OAuth application

  1. 在 github 中进行注册，进入 https://github.com/settings/profile

  2. 点击左侧 Developer settings

  3. 点击左侧Oauth Apps

  4. 点击 Register a new application

    注册完成之后，得到：Client ID 和 Client Secret

2.2 新建存放博客评论仓库

在 github 中建一个项目，专门用来存储你的博客评论

2.5 管理

评论显示在新建存放评论的仓库中的 issue 中。
